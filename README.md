## Описание программы

GUI-приложение «Training Planner» предназначено для планирования тренировок с фильтрацией по типу тренировки, сохранением данных в JSON.

---
---

## Требования для запуска

Что нужно установить на компьютер:

- Python 3.10 или выше
- Библиотеки: `pip install Tkinter

---

## Как запустить

https://github.com/Natali743/Planner.git
________________________________________
Как пользоваться
При запуске появляется окно, которое содержит поля для ввода даты, типа тренировки и длительности в мин.
По кнопке "Добавить тренировку", данная запись отражается в поле внизу окна.
В приложении есть возможность отфильтровать записи по следующим параметрам: по типу тренировки, по дате.
Есть возможность сброса фильтра.
Предусмотрена возможность сохранения данных в файл json.
